# slackee

`slackee` export all emoji from Slack.


## How to Use

Install from `npm`.

```bash
npm install slack-emoji-exporter
```

Setup environment variables `SLACK_API_TOKEN`.

```bash
export SLACK_API_TOKEN='xoxb-XXXXXXXXXXXX-XXXXXXXXXXXXXXXXXXXXXXXX'
```

Get emoji.

```bash
slackee -d -o output_dir
```

Show helps.

```bash
slackee --help
```
